# safennova-landing

Landing page de SafeNova — Gestión HSE para sitios industriales de alto riesgo.

**Live:** https://safennova-landing.vercel.app
**Repo:** https://github.com/Gigisanta/safennova-landing

## Stack

- Single HTML file (standalone, no build step)
- Google Fonts: Bricolage Grotesque + DM Sans + JetBrains Mono
- Lucide Icons via CDN
- CSS animations + IntersectionObserver

## Deploy

Push a `main` → deploy automático a Vercel.

```bash
git push origin main
```
